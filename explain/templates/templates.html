Template will consist of html files. To create a template for your view. you have tow ways First way: create templates folder in you app then create inside that templates folder. create folde by app name. Example: Return render(request, 'blogs/search_results.html', context) Search_results.html should exists in blogs/templates/blogs/search_results.html Second: Create folder in same level with project root. Structure should be like that.
